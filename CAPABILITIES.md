Slack Web API CLI. Call any Slack API method dynamically with flag-based or JSON input.
- Send, update, and delete messages; manage reactions and threads
- Channel operations: create, archive, invite/remove members, set topic/purpose
- User and user group lookups
- File uploads and sharing
- Search messages and files across the workspace
- Pagination support for large result sets (--paginate)
- Dry-run mode to preview requests without calling the API (--dry-run)
- Built-in method discovery: list-methods, describe <method>
